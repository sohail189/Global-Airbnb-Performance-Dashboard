🌍 Air Quality Analysis Dashboard
📌 Project Overview

The Air Quality Analysis Dashboard is an interactive Power BI project designed to monitor and analyze environmental pollution data.

This dashboard transforms raw air quality data into meaningful insights that support environmental monitoring and data-driven decision-making.

🎯 Project Objectives

Monitor Air Quality Index (AQI)

Analyze pollutant concentration levels

Identify pollution trends

Detect high-risk environmental conditions

Provide interactive visual insights
